PoliceBot

Discord bot for enforcing server rules.

Uses:

  • kick members
  • timeout members
  • send alert messages in channels
  • customize server profile (e.g. set mod roles)
  • report offending messages to mods
  • specialized holding room system with built in records*
  • waiting room system for new users
  • customizable waiting room questions*
  • kick members from a specific channel, or unkick them
  • custom server stats*

*on request from developers
