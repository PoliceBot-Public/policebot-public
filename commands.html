Slash Commands

For Everyone
Command Description Who can use
/help get a list of bot commands Everyone
/serverpic get the server picture Server Members
/server_owner find the server owner Server Members
/feedback send feedback to the bot developers Everyone
report report a message to mods* Server Members
*requires setup (see /profile init)

Many of the commands require server configuration. To configure your server, use /profile init. The options are as follows: Waiting room: the channel where new members go. Should be visible only to mods and the people with the new member role. New member role: the role that new members will get. Should only be able to see the waiting room. Mod role: the role for your mods to have. This role can access the moderation commands.
